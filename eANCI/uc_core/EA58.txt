disability mobility service, allow user to request mobility and transport services for disabled individuals, user, social services office. user selects mobility and transport service for disabled, system displays form for requester and recipient data entry, user enters their data and third-party details if needed specifying relationship, system retrieves requester data, recipient demographic data if different, recipient family unit information, user specifies recipient psycho-physical conditions with attached documents (disability certificate from local health authority, health records from physician, pension provider certificate, isee model) and possible social worker appointment date then submits form, system retrieves existing social assistance services recipient uses, forwards request to social services office and notifies user of submission, user must be successfully logged in and selected social assistance functionality. exit conditions include system detecting form errors invoking invalid data use case, user returning to previous screen, system detecting unexpected errors invoking service unavailable use case.
