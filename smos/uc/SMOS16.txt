Name: DeleteClass  
Actors: Administrator  
Description: Deletes a class from the archive  
Preconditions:  
• The user is logged into the system as an Administrator  
• The user has executed the use case "ViewClassDetails"  
• The user views the detailed information of a class  
• The user clicks the "Delete" button  

Event Sequence  
User  
System  
1. Deletes the class from the archive  
2. Displays the updated list of classes  

Postconditions:  
• The user has deleted a class  
• Connection to the SMOS server is terminated
