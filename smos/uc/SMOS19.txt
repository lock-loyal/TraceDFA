Name: ViewAddressDetails  
Actors: Administrator  
Description: Allows viewing the details of an address  
Preconditions:  
• The user has already logged in  
• The user has already executed the use case "ViewAddressList" and the system is displaying the list of addresses  
• The user clicks on the "show address details" button  

Sequence of Events  
User  
System  
1. Displays the screen with the details of a single address, specifically: address name.  

Postconditions:  
• The user views detailed information related to a single address.  
• Connection to SMOS server terminated Connection to SMOS server terminated
