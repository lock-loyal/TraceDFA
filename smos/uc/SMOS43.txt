Name: DeleteNote  
Actors: Administrator  
Description: Deletion of Disciplinary Notes  
Preconditions:  
• The user must be logged into the system as an Administrator.  
• The user has executed the "ViewNoteDetails" use case, and the system is displaying the details of a note.  
• The user clicks the "Delete" button.  

Sequence of Events  
User | System  
-----|-------  
1. The system sends an *errata corrige* notification to the student’s parents, then deletes the note’s data from the archive.  

Postconditions:  
• The note has been deleted, and the system has sent the notification to the parents. The system returns to the registry screen.  
• The administrator aborts the operation.  
• Connection to the SMOS server is interrupted.
