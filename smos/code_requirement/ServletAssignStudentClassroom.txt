- assign student to classroom, allows the administrator to assign or update a student's classroom association for a specific academic year, administrator. administrator selects student and academic year, system checks existing classroom assignment, removes previous association if exists, assigns new classroom if specified. student must exist and classroom must be valid for the academic year.  
- remove student classroom association, enables the administrator to disassociate a student from their current classroom, administrator. administrator selects student and academic year, system verifies existing classroom association, removes link between student and classroom. student must have active classroom assignment in specified academic year.
