- configure exam registration deadlines, allows the administrator to set minimum and maximum days allowed before an exam for student registration, administrator. administrator inputs the days in system configuration, system enforces these constraints during exam booking processes.  
- manage report storage path, enables the administrator to define the directory path for saving generated report pdf files, administrator. administrator specifies the storage location, system saves all reports to the configured directory.  
- configure email notification server, allows the administrator to set smtp server details for system-generated email alerts, administrator. administrator enters server address, system uses it to send automated emails for registrations and notifications.  
- set data import directory, allows the administrator to specify the upload directory for importing student and class data files, administrator. administrator defines the path, system processes and stores uploaded files in the designated location.
