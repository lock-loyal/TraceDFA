- view disciplinary note details, allows the administrator to view detailed information of a specific disciplinary note, administrator. administrator selects a note from the list, system displays note details including date, associated teacher, and description.  
- update disciplinary note, enables the administrator to modify an existing disciplinary note's information, administrator. administrator edits note details such as date, teacher, and description, clicks update button, system validates mandatory fields and saves changes, redirects to note list. mandatory fields: date, teacher, description.
