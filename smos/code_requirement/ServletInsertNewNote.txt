- insert a disciplinary note, allows the administrator to add a disciplinary note for a student, administrator. administrator selects student, system displays form for entering note details including date, teacher, and description, administrator fills form and submits, system saves note and redirects to student's note list. mandatory fields must be filled.
- view student disciplinary notes, displays list of disciplinary notes associated with a student, administrator. administrator selects student from list, system retrieves and displays all notes related to selected student including date, teacher, and description.
- access control for note management, restricts note creation and modification to administrators, administrator. non-administrator users attempting to access note management functions receive error message and are redirected to error page.
- handle note creation errors, ensures valid input data during note creation, administrator. system validates mandatory fields like date and description, displays error message if validation fails and returns to note entry form.
