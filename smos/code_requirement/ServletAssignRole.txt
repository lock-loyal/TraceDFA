- assign user roles, allows the administrator to assign or remove roles for a user, administrator. administrator selects target user and checks/unchecks role checkboxes, system updates user's role assignments, administrator cannot remove administrator role if target user is the last remaining administrator.
- manage role permissions, enables modification of user access levels through role assignments, administrator. administrator navigates to user management interface, selects user, modifies role selections from available options, system validates changes against administrative constraints.
- modify user privileges, updates system access permissions via role configuration, administrator. administrator accesses user profile, adds/removes roles through checkbox selections, submits changes, system enforces rule preventing removal of sole administrator account's admin privileges.
- configure system roles, handles assignment and revocation of functional permissions to users, administrator. administrator selects roles from available list for specific user, system applies selected roles while ensuring at least one administrator remains active in the system.
