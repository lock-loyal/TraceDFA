- view student report card, allows administrator or director to view a student's grade report for a selected academic year and term, administrator. administrator selects student, academic year, and term, system retrieves and displays grades for written, oral, and laboratory assessments from the database. student id, academic year, and term must be valid.
- display grade report, system presents a student's grades organized by assessment type and academic period, director. director accesses report management section, selects student and academic parameters, system displays report page with grades and student details. user must have director privileges to access functionality.
- access control for report viewing, restricts grade report access to authorized administrative roles, administrator. system verifies user role during report retrieval, displays error if unauthorized user attempts access. only administrators or directors can view reports.
