- manage student grades, allows the teacher or administrator to input and update student grades for specific courses, teacher/administrator. teacher/administrator enters grades for written, oral, and laboratory components within 0-10 range, system validates and saves grades, grades must be between 0 and 10.  
- view grade breakdown, enables students or parents to view detailed grade components for a course, student/parent. user selects academic year and semester, system displays written, oral, and laboratory grades with corresponding scores.  
- track academic progress by period, allows system to organize grades based on academic year and semester, administrator. administrator assigns grades to specific academic years and semesters when inputting data, system groups grades by these parameters for reporting.
