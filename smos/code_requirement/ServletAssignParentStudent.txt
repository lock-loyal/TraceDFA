- assign parent to student, allows the administrator to link a parent to selected students, administrator. administrator selects students from list and submits form, system updates parent association for selected students, administrator must have valid privileges and student must exist.
- remove parent from student, allows the administrator to unlink a parent from selected students, administrator. administrator deselects students from list and submits form, system removes parent association for deselected students, administrator must have valid privileges and student must exist.
